HTML Tags Documentation

Tag Name Formatted Tag Description
Heading <h1>...</h1> Defines the largest heading level.
Paragraph <p>...</p> Defines a paragraph of text.
Table <table>...</table> Defines a table layout for data.
Anchor <a href="...">...</a> Creates a hyperlink to another document or location.
Image <img src="..." alt="..."> Embeds an image into the document.
